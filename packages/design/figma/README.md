# Figma

## Overview

This package is designed specifically for generating files to upload into Figma. It is not intended for use in the codebase and is primarily utilized during the initial setup of the Figma project.

## Figma File

The Figma file includes all the essential components required for the project, ensuring a cohesive design system. It is regularly updated to reflect the latest changes and improvements, with each version clearly labeled for easy reference. This allows designers and developers to stay aligned with the most current design specifications and assets.
