# Design

This folder contains the following design-related packages:

- [Figma](figma/README.md): Tools and resources for Figma design.
- [Tailwind](tailwind/README.md): Tailwind CSS utilities and configurations.
- [Theme Builder](theme-builder/README.md): A tool for creating and managing themes.
- [Token Utils](token-utils/README.md): Utilities for managing design tokens.
- [Tokens](tokens/README.md): Design tokens for consistent styling.
- [Tokens Builder](tokens-builder/README.md): A tool for building and managing design tokens.
