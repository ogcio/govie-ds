# Wagtail example

### Setup env

```
mkdir env
python3 -m venv env
```

### Activate env

```
source env/bin/activate
```

## Getting Started

### Install requirements

```
pip3 install -r requirements.txt
```

### Database migrations

```
python3 manage.py migrate
```

### Create superuser

```
python3 manage.py createsuperuser
```

### Run server

```
python3 manage.py runserver
```
