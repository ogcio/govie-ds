# Cookie Banner

The Cookie Banner component is used to inform users about the use of cookies on a website and to obtain their consent for non-essential cookies.
It provides options for users to accept all cookies or only necessary cookies.

## Usage

A cookie banner is required whenever your application or website manages user information or tracks user behaviour.
This includes scenarios where cookies or similar technologies are used to collect analytics, personalise content, or store user preferences.
Informing users and obtaining their consent is essential for compliance with privacy regulations and for building user trust.

According to GDPR legislation, the consent acquired from users must be stored securely and in a way that can be audited if required.
Ensure your implementation records the user's consent decision and provides a mechanism to retrieve or update this information if necessary.
