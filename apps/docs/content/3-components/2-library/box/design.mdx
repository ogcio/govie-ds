# Box

Box is the simplest layout primitive in the design system — a plain `<div>` wrapper that signals design-system membership. All styling is applied through `className` with `gi-*` Tailwind utilities. Use Box instead of bare `<div>` elements to make design-system adoption visible and consistent across your codebase.

## Usage

Use Box as a design-system-native replacement for bare `<div>` elements. It provides no layout behaviour of its own — all styling comes from `className` with `gi-*` Tailwind utilities.

### Do

- Do use Box instead of bare `<div>` elements for design-system wrappers.
- Do apply styling through `className` with `gi-*` Tailwind utilities.
- Do use Box for simple spacing, backgrounds, and sizing wrappers.

### Don't

- Don't use Box when you need flex layout — use Stack instead.
- Don't add `role` or ARIA attributes directly — use Stack for landmark containers.
- Don't use inline styles — use `gi-*` Tailwind utilities via className.
