# Spinner

A spinner is a rotating circle that indicates the status or state of completion for a process that's part of a user flow, such as verifying personal information or sending a request. The circles can indicate that a process is underway.

### Do's

- Use spinners for short waits (under 10 seconds).
- Provide context, e.g., “Loading your profile…”.
- Place the spinner prominently with sufficient contrast.
- Prefer skeletons or progress bars for longer tasks.

### Don'ts

- Don't show multiple spinners on the same context.
- Don't leave users waiting without timeout/error handling or a retry option.
