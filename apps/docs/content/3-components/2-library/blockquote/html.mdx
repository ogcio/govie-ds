# Blockquote

The blockquote HTML component is used to highlight content that is quoted from another source. It provides semantic meaning and visual distinction for quotations, often accompanied by a citation. Use it to emphasize statements, legal information, testimonials, or referenced material.

<Section current="html">

<ComponentPreview>
  <Blockquote cite="https://example.com/source">
    It can take up to 8 weeks to register a lasting power of attorney if there
    are no mistakes in the application.
  </Blockquote>
</ComponentPreview>
