# Blockquote

<InsetText>
  `Blockquote` is deprecated. Use `InsetText` instead. Blockquote will be
  removed in version 2.
</InsetText>

`Blockquote` is kept only for backward compatibility. New code should use `InsetText`.

<Section current="design">
<ComponentPreview>
  <Blockquote cite="https://example.com/source">
    It can take up to 8 weeks to register a lasting power of attorney if there
    are no mistakes in the application.
  </Blockquote>
</ComponentPreview>

## Use `InsetText` (recommended)

<ComponentPreview>
  <InsetText cite="https://example.com/source">
    It can take up to 8 weeks to register a lasting power of attorney if there
    are no mistakes in the application.
  </InsetText>
</ComponentPreview>

Use `InsetText` to visually differentiate a block of text from the content that surrounds it.
If the content is a quote, you can still provide `cite`.

</Section>
