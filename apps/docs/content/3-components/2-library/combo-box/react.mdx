# Combo Box

The **Combo Box** component combines an input field with a dropdown list, allowing users to either select from a predefined set of options or type their own value.  
It supports both controlled and uncontrolled modes, multi-select capabilities, and optional chips to display selected items.  
You can customise its behaviour and appearance via props and integrate it seamlessly in forms.
