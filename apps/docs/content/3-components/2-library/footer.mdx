## Usage

Use the footer at the bottom of every page of your service.

### Adding Links

You can add links to:

- Privacy notice
- Accessibility statement
- Cookies page
- Terms and conditions
- Other language options
- Help content

Use âPrivacyâ, âAccessibilityâ, âCookiesâ, and âTerms and conditionsâ for the link text.

### Adding Secondary Navigation

Only add secondary GOV.IE navigation if youâre creating a GOV.IE service, and you want users to be able to navigate away from the service to somewhere else on the GOV.IE website. For example, you probably donât want to encourage a user to navigate away from a linear, form-type service.

### Footer Component Slots

The footer component now accepts 3 customizable slots:

- **Primary Slot**: For essential links such as the privacy notice, accessibility statement, terms and conditions, etc.
- **Secondary Slot**: For secondary navigation, especially if you need to add other relevant links or additional GOV.IE content.
- **Utility Slot**: For supplementary items such as language options, help content, or other utilities.

### Footer Props

- **primarySlot** (optional): A node rendered in the primary section of the footer. Typically used for key links or content.

- **secondarySlot** (optional): A node rendered in the secondary section of the footer. Useful for secondary navigation or additional content.

- **utilitySlot** (optional): A node rendered in the utility area of the footer. Often used for legal links, language selectors, or settings.

- **logo** (optional): An object of type `LogoProps` representing the logo displayed in the footer.

- **className** (optional): A string for applying custom CSS classes to the footer container.

- **dataTestid** (optional): A string used as a test identifier for testing frameworks.

### Best Practices

- Always use the footer at the bottom of every page.
- Ensure the primary slot includes links to essential content like privacy, accessibility, and terms.
- Only use secondary navigation for GOV.IE services and avoid overloading the footer with unnecessary links.
- Make sure the utility slot is used sparingly for items that are not essential for navigation but are still useful for the user.

### When to Use This Component

Use the footer when you need to provide important links or secondary navigation at the bottom of your service pages. This is a standard design element for providing users with easy access to essential information and resources.

### When Not to Use This Component

Avoid using the footer if you donât have enough content for it, or if adding a footer could overwhelm or distract users from the main content of the page. Also, refrain from using it for complex navigation schemes or as a place for redundant content that doesn't contribute to user experience.
