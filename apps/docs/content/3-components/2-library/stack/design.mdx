# Stack

Use the Stack component to arrange items vertically or horizontally with consistent spacing, alignment, and responsive layout options. It's ideal for stacking related elements, like buttons, cards, or form fields, in a flexible and organised way.

## Usage

### Do

- Do keep spacing consistent by using the `gap` prop instead of ad-hoc margins.
- Do use responsive objects (e.g., `{ base: 'column', md: 'row' }`) for layout changes.
- Do align and distribute items using `itemsAlignment` and `itemsDistribution` for clarity.

### Don't

- Don't mix manual margins and `gap` for the same itemsâ€”pick one spacing system.
- Don't force Stack to behave like a grid when a Grid layout is more appropriate.
- Avoid adding dividers everywhere; use hasDivider only when it genuinely helps readability.
