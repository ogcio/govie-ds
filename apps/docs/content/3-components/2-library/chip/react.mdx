# Chip

The Chip component in React is used to display compact, contextual information such as tags or filters. It accepts a `label` prop to define the content and optionally an `onClose` callback if the chip should be dismissible. Chips are fully accessible by default and support keyboard interactions like `Enter` and `Space` to trigger dismissal when focused.

## i18n Keys

The Chip component supports localisation through i18n keys in React, [find i18n setup here](/components/setup-guides/react/#internationalisation-i18n-guidelines). Below is the list of keys and their default text:

```json
{
  "chip.removeChip": "Remove chip",
  "chip.label": "Chip: {{label}}"
}
```

These keys represent visible text elements. You can customise their translations for each locale to enable multilingual support.

- Ensure placeholders like `{{ label }}` is preserved when translating.
