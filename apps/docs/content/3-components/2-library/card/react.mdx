## i18n Keys

The Card component supports localisation through i18n keys in React, [find i18n setup here](/components/setup-guides/react/#internationalisation-i18n-guidelines). Below is the list of keys and their default text:

```json
{
  "card.cardTitle": "Card {{title}}",
  "card.actionLink": "Action link: {{children}}",
  "card.actionButton": "Action button: {{children}}",
  "card.subTitle": "Subtitle: {{subTitle}}",
  "card.tag": "Tag: {{tag}}"
}
```

These keys represent visible text elements. You can customise their translations for each locale to enable multilingual support.

- Ensure placeholders like `{{ title }}`, `{{ subTitle }}`, and `{{ tag }}` are preserved when translating.
