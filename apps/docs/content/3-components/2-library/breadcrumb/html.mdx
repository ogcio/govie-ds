# Breadcrumbs

The Breadcrumbs component displays the user's current location within a site's hierarchy and provides easy navigation to previous levels. It supports icons, custom links, and ellipsis to manage complex paths while maintaining accessibility and clarity.

<Section current="html">

<ComponentPreview>
  <Breadcrumbs>
    <BreadcrumbLink href="/home">Home</BreadcrumbLink>
    <BreadcrumbEllipsis />
    <BreadcrumbLink href="/docs">Documentation</BreadcrumbLink>
    <BreadcrumbCurrentLink href="/travel">Travel abroad</BreadcrumbCurrentLink>
  </Breadcrumbs>
</ComponentPreview>
