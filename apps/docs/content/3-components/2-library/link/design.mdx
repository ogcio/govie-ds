# Link

Links are blue and underlined by default. If your link is at the end of a sentence or paragraph, make sure that the linked text does not include the full stop.

## Usage

Use the Link component to create consistent, accessible, and visually appealing hyperlinks. It handles external links, disabled states, icons, and different visual styles like underline and color inheritance.

### Do

- Use for simple navigational hyperlinks (internal or external).
- Use **external** to properly mark and secure external links (adds rel="noopener noreferrer" and target="_blank").
- Use **iconStart** and **iconEnd** to enhance clarity with visual indicators.
- Use **noUnderline**, **noVisited**, or **noColor** to match custom style requirements.
- Use **asChild** to wrap custom components or slots while preserving semantics.

<ComponentContainer>
    <Link href="#" external iconStart="arrow_back">Back to Docs</Link>
</ComponentContainer>

### Don't

- Don't place block-level elements like ```<div>``` or ```<p>``` directly inside the Link — wrap content in ```<span>``` or use asChild.
- Don't apply complex visual interactions (like transitions or animations) beyond its scoped styles — use a custom component instead.
- Don't override link behaviour purely with ```onClick```  — use a Button if it doesn’t navigate.

<Link href="#">
  <p className="gi-text-lg">Link with paragraph</p>
</Link>
