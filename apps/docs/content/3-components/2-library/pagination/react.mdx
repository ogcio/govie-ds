# Pagination

The Pagination component provides a user interface for navigating through multiple pages of content. It dynamically adjusts the number of page buttons shown based on screen size and the total number of pages, using ellipses (`...`) when necessary to keep the layout compact. On small screens, it switches to a compact view with only previous/next buttons and a current page indicator.

## i18n Keys

The Pagination component supports localization through i18n keys in React, [find i18n setup here](/components/setup-guides/react/#internationalisation-i18n-guidelines). Below is the list of keys and their default text:

```json
{
  "pagination.previous": "Previous",
  "pagination.next": "Next",
  "pagination.page": "Page {{currentPage}} of {{totalPages}}",
  "pagination.goToPage": "Go to page {{page}}",
  "pagination.goToPrevious": "Go to previous",
  "pagination.goToNext": "Go to next"
}
```

These keys are used for screen reader accessibility (aria-label) and visible button text.
- You can override these translations per locale to support multilingual experiences.
- Ensure placeholders like `{{ currentPage }}`, `{{ totalPages }}`, and `{{ page }}` are preserved when translating.
