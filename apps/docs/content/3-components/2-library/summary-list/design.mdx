# Summary List

This component provides a way to present structured information in rows of labels and values. It is commonly used for displaying a user’s details before confirmation or submission.

## Usage

Use the summary list component to display structured information in a way
that is easy to scan. Each piece of data is shown as a row with a label,
value, and optional action.

### Do

- Use this component for presenting **reviewable information** (e.g., before submitting a form).
- Keep labels short and descriptive.
- Include actions only when users can take action on the item.
- Use line breaks for multi-line content like addresses.

### Don't

- Don’t overload a row with too much information, split into multiple rows if needed.
- Don’t use this component for tabular data, use a table instead.
- Don’t include actions that don’t directly relate to the value.
