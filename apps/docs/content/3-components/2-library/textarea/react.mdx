# Textarea

The Textarea component allows users to input multi-line text. It's ideal for longer content such as messages, feedback, or descriptions. It supports customizable dimensions with `rows` and `cols`, and character limits using `maxChars`.

### With Max Chars (Deprecated)

> **Note:** The `maxChars` prop is deprecated. Use the `CharacterCount` component instead for better support with uncontrolled forms and React Hook Form. The `maxChars` prop works correctly in controlled mode (when passing `value`), but in uncontrolled mode the character count may not update correctly after `reset()`.

## i18n Keys

The Textarea component supports localisation through i18n keys in React, [find i18n setup here](/components/setup-guides/react/#internationalisation-i18n-guidelines). Below is the list of keys and their default text:

```json
{
  "textarea.remainingChars": "You have {{remainingChars}} characters remaining"
}
```

These keys represent visible text elements. You can customise their translations for each locale to enable multilingual support.

- Ensure placeholders like `{{ remainingChars }}` is preserved when translating.
