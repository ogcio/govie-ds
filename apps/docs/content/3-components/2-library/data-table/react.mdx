# Data Table

Design System component library provides a reusable Data Table component built with React. This component is designed to display and manage large sets of information in a structured format, allowing users to easily scan, filter, and manipulate data.
In the page you can find an example on how to create a Data Table using React components with [Tanstack table library](https://www.figma.com/design/ZEqfpXhocdZs1K5YfsE5ws/Design-system?node-id=7649-344).
