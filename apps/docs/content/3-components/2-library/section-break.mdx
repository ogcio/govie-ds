# Section Break

## Variants

### Default

<ComponentContainer>
  <SectionBreak size="md" />
</ComponentContainer>

You can use the Section Break component to create a thematic break between sections of content.
The section break has modifiers for different size margins.

## Usage

### SectionBreak Props

- **size** (optional): Controls the height or spacing of the section break. Accepts one of the following values:

  - `'sm'`: Small
  - `'md'`: Medium
  - `'lg'`: Large
  - `'xl'`: Extra Large

- **color** (optional): A string representing the color of the section break. Can be any valid CSS color value.

- **dataTestid** (optional): A string used as a test identifier for testing frameworks.
