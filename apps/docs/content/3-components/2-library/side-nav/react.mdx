# Side Navigation

Side Navigation (SideNav) is used to organize and access major sections of an application. It provides persistent and hierarchical navigation, making it ideal for dashboards or multi-section layouts.

<Section current="react">

<ComponentPreview className="gi-not-prose w-full block">
  <SideNav value="projects-active">
    <SideNavItem value="dashboard" label="Dashboard" />
    <SideNavItem expandable value="team" label="Team">
      <SideNavItem value="team-members" label="Members" />
      <SideNavItem value="team-permissions" label="Permissions" />
    </SideNavItem>
    <SideNavItem open expandable value="projects" label="Projects">
      <SideNavItem value="projects-active" label="Active" href="#" />
      <SideNavItem value="projects-archived" label="Archived" href="#" />
    </SideNavItem>
    <SideNavItem value="settings" label="Settings" />
  </SideNav>
</ComponentPreview>
