### Do

- Use clear, concise headers that describe the column content.
- Align numeric data to the right for readability.
- Keep table content consistent in format (e.g., same date or currency style).
- Provide captions for accessibility and context.

### Don’t

- Don’t use tables for layout purposes.
- Don’t mix unrelated data types in the same column.
- Don’t leave columns or rows empty for spacing purposes.
- Don’t use overly long text without wrapping; it can break the table layout.
- Don’t use very small font sizes to fit data—maintain readability.
- Don’t overload a table with too many columns.
