# Section Break

> **Deprecated:** This component is deprecated. Use [`<Divider>`](/components/library/divider/react/) instead. Divider provides the same visual separation without built-in margins — control spacing through your parent layout.

You can use the Section Break component to create a thematic break between sections of content.
The section break has modifiers for different size margins.
