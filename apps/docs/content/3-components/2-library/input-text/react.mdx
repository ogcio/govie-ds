# Input Text

The Input Text component allows users to enter and edit text. It supports various input types, prefixes/suffixes, icons, and a clear button.

      text: 'Label',

  >
    <InputText data-testid="input-text-id" id="input-text-id" />
  </FormField>
</ComponentPreview>
