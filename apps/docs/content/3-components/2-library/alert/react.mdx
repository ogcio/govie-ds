# Alert

The Alert component is used to display important feedback messages to users. It supports different variants like success, warning, info, and danger, and may be dismissible.

## i18n Keys

The Alert component supports localisation through i18n keys in React, [find i18n setup here](/components/setup-guides/react/#internationalisation-i18n-guidelines). Below is the list of keys and their default text:

```json
{
  "alert.dismissAlert": "Dismiss Alert"
}
```

These keys represent visible text elements. You can customise their translations for each locale to enable multilingual support.
