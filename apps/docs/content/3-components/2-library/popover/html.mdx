# Popover

The Popover component is a versatile UI element used to display contextual information or actions when a user interacts with a specific trigger element. It leverages [Popper.js](https://popper.js.org/docs/v2/) for precise positioning and offers built-in functionality for managing its open/close state through external clicks and keyboard interactions.

## Programmatically Close Popover

You can programmatically close the Popover by dispatching a `closePopover` event. This is useful when interacting with other components (e.g., when a dropdown option is selected).
To close the Popover from an external element (like an input or list item), dispatch a `CustomEvent` named `closePopover`. Make sure it bubbles so it reaches the Popover container:

```ts
const closePopoverEvent = new CustomEvent('closePopover', {
  bubbles: true,
});
this.input.dispatchEvent(closePopoverEvent);
```
