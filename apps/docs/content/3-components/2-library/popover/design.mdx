# Popover

The Popover component displays floating content relative to a trigger element. It is used for dropdowns, menus, or any contextual UI element that should appear above other interface elements.

## Usage

Use popovers to provide temporary, contextual interactions or information. Popovers should be clearly linked to a trigger and should close on outside click or Escape.
