## Usage

### Best Practices

- Use the Accordion for organising large chunks of content that can be expanded and collapsed.
- Keep Accordion items focused and concise, with clear labels and descriptions.
- Use iconStart to make it easier for users to visually distinguish different items.

### When to use this component

- Organising content where only one section should be visible at a time.
- Presenting FAQs, help topics, or other collapsible content.

### When not to use this component

- For static content that doesnâ€™t need to be collapsed.
- For content that requires persistent visibility.
