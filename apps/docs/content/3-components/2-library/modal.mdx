## Usage

<ComponentPropertiesTable componentId="modal" />

### Modal Props

- **className** (optional): A string for custom styling of the modal.
- **closeButtonLabel** (optional): A string defining the label for the close button, improving accessibility.
- **triggerButton** (required): A `React.ReactElement` that triggers the modal when clicked.
- **children** (required): The modal content, structured using `ModalChildren`.
- **startsOpen** (optional): A boolean indicating whether the modal should be open by default. Defaults to `false`.
- **closeOnClick** (optional): A boolean that, when `true`, allows closing the modal via the close button or by clicking on the overlay. Defaults to `true`.
- **closeOnOverlayClick** (optional): A boolean that enables closing the modal when clicking outside its content (on the overlay). Defaults to `true`.
- **dataTestId** (optional): A string used for testing purposes.
- **size** (optional): Defines the size of the component. Accepts one of the following values:
  - `'sm'`: Small size.
  - `'md'`: Medium size.
  - `'lg'`: Large size (default).

### ModalWrapper: An Alternative for More Control

`ModalWrapper` includes all properties from `Modal`, except for **triggerButton**, and provides additional control over modal visibility. It is useful when managing the modal state externally.

#### Additional ModalWrapper Props
- **isOpen** (required): A boolean indicating whether the modal is open.
- **onClose** (required): A function that handles closing the modal.
- **position** (optional): Defines the modal's position, using `ModalPositions`.
- **closeButtonSize** (optional): Sets the size of the close button, using `ButtonSize`.

Using `ModalWrapper` allows direct control over the `isOpen` state, making it ideal for programmatically handling modal visibility.

### Best Practices

### When to use this component

Modals are best to use when trying to isolate an action by the user. By using a modal you capture the focus of the user and this allows to remove secondary distractions.

### When not to use this component

Don't use modal dialogs for elements that should be linkable or searchable.
