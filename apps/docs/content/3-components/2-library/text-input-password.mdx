# Text Input Password

The `TextInputPassword` component provides a secure password input field with an integrated visibility toggle button. This allows users to switch between masked and unmasked input modes.

## Usage

### Best Practices

### When to Use This Component

- **Secure Input Fields**: Use this component for password fields in login, signup, and authentication forms.
- **Toggleable Password Visibility**: Ideal for user convenience, allowing users to verify their password input.
- **Error Handling**: Works well with validation messages for incorrect passwords.
- **Accessibility Support**: Includes `aria-label` for visibility toggling to enhance screen reader usability.

### When Not to Use This Component

- **Non-Sensitive Inputs**: If the field does not contain sensitive information, use a standard `TextInput` instead.
