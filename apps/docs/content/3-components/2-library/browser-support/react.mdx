# Browser Support

The BrowserSupport component detects the user's browser and shows a dismissible warning when it falls below the supported policy. It's silent on supported browsers and surfaces a link to the official supported browsers page.

<Section current="react">

<ComponentPreview>
  <BrowserSupport forceShow />
</ComponentPreview>

## How it works

- Detects browser on mount.
- Compares result against the internal policy
- If **supported** and `forceShow` is `false`, renders nothing.
- If **unsupported** or `forceShow` is `true`, renders a `warning` `<Alert>`.

### Performance & UX

- No layout shift: banner appears in a reserved area of your layout (recommend placing near top).
- Silent on supported browsers to reduce banner fatigue.

### Server rendering

- The component is **client-only**. It does not SSR render the alert; it evaluates on the client and renders only if needed.
