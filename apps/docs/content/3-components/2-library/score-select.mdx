# Score Select

Score Select is used to collect a rating score from users, typically in the form of Net Promoter Score (NPS). It displays a horizontal row of buttons representing numeric values from 0 to 10, and includes labels on either side to help frame the scale.

## Usage

### ScoreSelect Props

- `label` (required): The main label above the button group.
- `hint` (optional): A brief description or instruction shown under the label.
- `name` (required): The name attribute used for form submission.
- `defaultValue` (optional): The initial selected score.
- `options` (required): An array of `{ label, value }` pairs, typically from 0 to 10.
- `leftLabel` (optional): The text that appears on the left of the score range (e.g., "Not Likely").
- `rightLabel` (optional): The text that appears on the right of the score range (e.g., "Extremely Likely").
- `size` (optional): Size of the buttons. Options include `small`, `medium`, `large`. Default is `medium`.
- `onChange` (optional): Callback when the selected score changes.

### Best Practices

#### When to use this component

- Use the Score Select component when you need a quick and intuitive way to collect user ratings, such as:
- Net Promoter Score (NPS) surveys.
- Satisfaction ratings.
- Feedback forms where numeric rating is required.

#### When not to use this component

- Do not use if:
- You require text-based answers or qualitative feedback.
- You need more than one score group in a single input.
- A slider or range input would be more appropriate for continuous values.
